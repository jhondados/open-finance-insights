# 💰 Open Finance Insights

> Agregador de dados Open Finance que analisa gastos, gera insights financeiros personalizados e recomenda produtos com IA generativa.

**🌐 Demo ao vivo (web app interativo):** https://jhondados.github.io/open-finance-insights/

## 📊 Resultados-chave (KPIs)
**142K** Usuários · **R$ 380/mês** Economia média · **73%** Engajamento · **68** NPS

## 🧠 Tecnologias de IA
GenAI · ML · Recommendation · RAG

## ✨ O que faz
- Categorização automática de gastos
- Insights gerados por GenAI
- Recomendação de produtos
- Score de saúde financeira

## 🗄️ Fonte de dados real
[Open Finance Brasil](https://openfinancebrasil.org.br)

## ⚙️ Como funciona
1. **Coleta de dados reais** — integração com Open Finance Brasil
2. **Processamento com IA** — GenAI · ML · Recommendation · RAG
3. **Dashboard & insights** — KPIs, gráficos e recomendações acionáveis
4. **Ação automatizada** — alertas, predições e decisões em tempo real
